# Prompt Construction System

How to build glitchcore prompts that produce real artifact logic, body transformation, palette hierarchy, and emotional consequence instead of generic neon slop.

## Purpose

This file defines a reusable system for constructing glitchcore prompts.

Its job is to help prompts produce images that are:
- Specific
- Artifact-driven
- Emotionally charged
- Visually coherent
- Palette-aware
- Body-centered
- Hybrid-friendly
- Resistant to drift

A good glitchcore prompt should not just say:
- Glitchy
- Distorted
- Neon
- Cyberpunk
- VHS
- Digital

That weak-ass language makes weak-ass outputs.

A strong prompt should specify:
- What kind of figure or identity image it is
- What artifact driver is leading
- What secondary systems support it
- What palette family is active
- What density level the image should operate at
- What emotional territory it occupies
- What kind of media failure is implied
- What must remain readable

---

## 1. Core Prompt Formula

A glitchcore prompt should usually contain these layers:

1. Subject / figure anchor
2. Image mode / framing
3. Primary artifact driver
4. Secondary artifact drivers
5. Palette energy family
6. Density level
7. Emotional flavor
8. Material / display feel
9. Retrieval path / preserved anchor
10. Anti-drift clause

That gives you enough structure to make the corruption meaningful without turning the prompt into random word diarrhea.

---

## 2. The Master Prompt Skeleton

Use this as the base template:

```
[subject / body / face / identity image]
treated as [subtype or image mode],
with [primary artifact driver] as the main transformation logic,
supported by [1–3 secondary artifact behaviors],
using [palette energy family] with [dominant hue / collision hue / accent puncture],
at [signal density level],
creating a mood of [emotional flavor words],
with a surface feel of [material / media / display language],
while preserving [retrieval path / readable body anchor],
and avoiding [main drift risks].
```

---

## 3. Layer-by-Layer Prompt Construction

### Layer 1 — Subject / Figure Anchor

This is the body the glitch happens to. Without this, the image can float off into pure abstract effect bullshit.

**Use:**
- Close-up portrait
- Frontal face
- Profile portrait
- Bust image
- Head-and-shoulders portrait
- Beauty closeup
- Glam figure
- Dancer mid-gesture
- Solitary standing figure
- Icon-like portrait
- Moving body
- Self-image
- Body silhouette
- Face with hands near mouth or cheeks

**Stronger variants:**
- Heavily stylized beauty portrait
- Emotionally charged close-up face
- Diva-coded bust portrait
- Moving body caught mid-turn
- Intimate profile image
- Screen-mediated self-portrait
- Body posed like a signal emergency
- Face as emotional broadcast surface

**Rule:** Say what kind of body/figure image this is before you start corrupting it.

---

### Layer 2 — Image Mode / Subcategory Choice

Choose which glitchcore branch is actually being used.

**Options:**
- Candy broadcast hallucination
- Pastel identity smear
- RGB phantom
- Composite fever dream
- CRT contour
- Ghost-frame body
- Raster meltdown
- Pixel ghost
- Text-screen heartbreak
- Surveillance apparition
- Candy-crash compression
- Glitter-signal overprint

**Example:**
- "treated as a candy broadcast hallucination"
- "rendered as a pastel identity smear"
- "approached as a ghost-frame body image"
- "built like a text-screen heartbreak portrait"

**Rule:** Name the family, or describe it with equivalent specificity.

---

### Layer 3 — Primary Artifact Driver

This is the most important part.

**Main options:**
- Channel split
- Temporal echo
- Scanline contour banding
- Raster tear
- Compression chew
- Pixel matrix visibility
- Overprint stacking
- Bloom contamination
- Noise veil
- Text/interface debris
- Capture degradation
- Color field collision

**Example phrases:**
- "with channel split as the main transformation logic"
- "driven primarily by overprint stacking and repeated facial copies"
- "built around quantized bloom and codec chew"
- "using temporal echo as the main body-distortion system"
- "structured by scanline contour banding"

**Rule:** One lead driver. Always.

---

### Layer 4 — Secondary Artifact Drivers

These deepen the image without replacing the lead.

**Good combinations:**
- Channel split + bloom + overprint
- Temporal echo + channel split + glow haze
- Compression chew + text debris + block shimmer
- Scanline contour + noise veil + phosphor bloom
- Text debris + overprint + subtitle ghosting

**Good phrasing:**
- "supported by bloom-heavy chromatic haze and low-opacity overprint copies"
- "with subtitle ghosting and compression scars as secondary contamination"
- "layered with sparkle-static, text debris, and mild packet-loss breakup"
- "supported by faint static veil and phosphor halo"

**Rule:** Usually 1–3 secondaries, not 8.

---

### Layer 5 — Palette Energy Family

This determines how the image feels emotionally and visually.

**Palette families:**
- Phosphor noir
- Hyperpop rupture
- Pastel signal melt
- Rainbow channel hallucination
- Toxic cathode candy
- Gloss-electric glam
- Codec candy crash

**Structure it like this:**
- "using a hyperpop rupture palette"
- "in pastel signal melt colors"
- "with gloss-electric glam chroma"
- "using codec candy crash palette logic"

**Better: add architecture:**
- "dominant hot pink and magenta"
- "cyan as collision hue"
- "white flare as accent puncture"
- "toxic lime intrusion"
- "lavender haze support"
- "coral mouth highlight"
- "icy cyan contour bands"

**Rule:** Don't just say "colorful." That means nothing.

---

### Layer 6 — Density Level

Choose how loaded the image should feel.

**Options:**
- Sparse signal
- Moderate signal
- Dense signal
- Overloaded signal
- Pile-up delirium

**Example phrases:**
- "at moderate signal density"
- "running at dense signal load"
- "pushed into overloaded broadcast pressure"
- "with pile-up delirium but preserved facial readability"

**Rule:** This is what stops the prompt from becoming too timid or too sludgy.

---

### Layer 7 — Emotional Flavor

This tells the corruption what it means.

**Emotional flavors:**
- Dreamy
- Seductive
- Haunted
- Ecstatic
- Lonely
- Confessional
- Bratty
- Glamorous
- Anxious
- Dissociative
- Panicked
- Cyber-romantic
- Overstyled
- Internet-broken
- Ritualistic
- Synthetic-tender
- Emotionally overfucked

**Example phrasing:**
- "creating a mood of glamorous instability and synthetic longing"
- "with a dreamy, coquettish, emotionally lacquered tone"
- "feeling bratty, overprocessed, and a little heartbroken"
- "conveying ecstatic color overload and unstable selfhood"

**Rule:** Emotion is not optional. Otherwise it becomes a dead tech demo.

---

### Layer 8 — Material / Media Feel

What kind of screen/body/surface is this happening in?

**Good options:**
- CRT phosphor
- Glossy digital surface
- Reposted screenshot mush
- Terminal-screen residue
- Bloom-heavy monitor glow
- Codec-bruised skin
- Toy-plastic shine
- Subpixel field
- Dead-soft export surface
- Sparkly overprint haze
- Scan-emissive contour skin
- Damaged feed texture

**Example phrasing:**
- "with a glossy, bloom-heavy digital surface"
- "rendered with dead-soft recompression mush and buzzing chroma edges"
- "carrying CRT phosphor striping and faint static grit"
- "with lip-gloss shine corrupted by codec bruise"

**Rule:** This is how the viewer feels the image as media.

---

### Layer 9 — Retrieval Path / Preserved Anchor

This is the part that must survive.

**Good anchors:**
- Readable eyes
- Lips
- Profile line
- Head silhouette
- One hand gesture
- Jaw contour
- Shoulders
- One central phrase
- One repeated body pose
- A strong face plane

**Example phrasing:**
- "while preserving readable eyes and mouth"
- "with the profile line still clearly recoverable"
- "keeping the silhouette and hand gesture intact"
- "preserving one strong facial anchor amid heavy artifacting"

**Rule:** No anchor = visual soup.

---

### Layer 10 — Anti-Drift Clause

A short phrase to stop the image from wandering into generic land.

**Good anti-drift clauses:**
- "not a clean cyberpunk editorial image"
- "not a simple RGB split filter"
- "not a beauty ad with decorative glitch"
- "not generic VHS nostalgia"
- "not empty rainbow moodboarding"
- "corruption must alter the face and image structure"
- "artifacts should feel embedded, not overlaid"
- "the image should remain hot but genuinely unstable"

**Rule:** This is the fence that keeps the goats out.

---

## 4. Short Prompt Formula

When you don't want the full cathedral, use this compact format:

```
[subject] + [subtype] + [primary driver] + [secondary drivers] + [palette family] + [density] + [emotion] + [anchor] + [anti-drift]
```

**Example:**
```
close-up glam portrait + candy broadcast hallucination + channel split + bloom and overprint + hyperpop rupture palette + dense signal + bratty glamorous heartbreak + preserve eyes and lips + not a clean editorial beauty image
```

That alone is already way better than "glitchy cyberpunk girl."

A frontal face portrait rendered as a pastel identity smear, driven by overprint stacking and repeated facial copies, supported by soft channel split and creamy bloom haze, in a pastel signal melt palette with dominant cotton-candy pink, lavender collision, and pearl-white highlight punctures, at moderate signal density, creating a dreamy, dissociative, romantically unstable mood, with velvety cosmetic softness and slightly corrupted beauty-surface glow, while preserving the eyes, lips, and cheek contours, not a generic dreamy filter or beauty ad.

## 6. Hybrid Prompt Construction

A lot of the best images are hybrids.

**Hybrid formula:**
```
[subject] treated as a hybrid of [subtype A] and [subtype B], led by [primary driver], supported by [secondary systems], using [palette family], at [density], creating [emotion], with [material feel], preserving [anchor], avoiding [drift].
```

**Example:**
```
A close-up diva portrait treated as a hybrid of glitter-signal overprint and text-screen heartbreak, led by overprint stacking, supported by subtitle debris, bloom haze, and soft channel split, using gloss-electric glam palette logic, at dense signal density, creating coquettish loneliness and glamorous instability, with lip-gloss shine and dead-screen text residue, preserving the eyes and mouth, not a beauty ad or graphic design poster.
```

## 7. Good vs Bad Prompt Language

**Weak:**
- Glitchy portrait
- Neon cyber face
- VHS girl
- Dreamy glitch edit
- Colorful digital distortion
- Retro futuristic beauty shot
- Moody RGB fashion portrait

**Better:**
- Beauty closeup transformed by channel split and candy-burn bloom
- Overprinted face portrait with repeated lips and pastel identity smear
- Packet-loss glam portrait with macroblock bruising and subtitle ghosting
- Scanline contour profile built from cyan stripes and white emission bands
- Glossy diva image corrupted by text debris and codec-soft chroma bleed

---

## 8. Summary Sentence

A strong glitchcore prompt specifies the figure, the artifact system, the palette logic, the density level, the emotional consequence, the material/media feel, and the preserved human anchor so the resulting image becomes truly re-authored by failure rather than merely decorated by effects.

---

## 9. Short Anchor Rule

Don't prompt for "glitch vibes." Prompt for what kind of body failure, what kind of color stress, what kind of media damage, and what kind of feeling.
