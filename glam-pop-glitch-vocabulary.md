# Glam-Pop Glitch Vocabulary

The emotional language of bright, seductive, hyperpop-influenced glitchcore.

## Purpose

This file defines the vocabulary for glam-pop glitchcore — the emotional language, descriptive terms, and aesthetic markers that make bright glitchcore distinct from dark or neutral glitch.

This is the part that helps the app stop going:
"make it colorful and glitchy"
and start going:
"seductive signal damage with sugary heartbreak undertones."

The vocabulary here is designed to make bright glitchcore emotionally specific and stylistically coherent.

---

## 1. Core Emotional Terms

### 1.1 Seductive

**Meaning:** Attractive in a way that draws the viewer in, alluring, magnetic

**Visual Markers:**
- Face remains attractive despite corruption
- Eyes and lips preserved
- Corruption feels like it's framing beauty, not destroying it
- Warm color palette (pinks, magentas)
- Soft glow or bloom

**Usage:**
```
"seductive candy broadcast hallucination"
"seductive with bratty undertones"
```

**Opposite:** Repulsive, off-putting, aggressive

---

### 1.2 Bratty

**Meaning:** Defiant, playful, cheap-beautiful, doesn't-care energy

**Visual Markers:**
- Compression damage visible
- "Internet-broken" quality
- Saturated but slightly wrong colors
- Feels like a screenshot reposted too many times
- One beauty zone preserved defiantly

**Usage:**
```
"bratty file damage"
"bratty with glamorous undertones"
```

**Opposite:** Polished, refined, careful

---

### 1.3 Sugary Heartbreak

**Meaning:** Sweet but sad, cotton-candy melancholy, romantic disappointment

**Visual Markers:**
- Pastel pinks and blues
- Soft, dreamy corruption
- Fragile, dissociative feeling
- Beautiful but slightly broken
- Lonely in a pretty way

**Usage:**
```
"sugary heartbreak aesthetic"
"sugary heartbreak with dreamy overprint"
```

**Opposite:** Bitter, harsh, angry

---

### 1.4 Synthetic Euphoria

**Meaning:** Artificial joy, chemically-enhanced happiness, hyperpop energy

**Visual Markers:**
- Hyper-saturated colors
- Electric cyan and magenta
- High energy, unstable
- Feels like it's about to burst
- Digital, processed happiness

**Usage:**
```
"synthetic euphoria mode"
"synthetic euphoria with channel split"
```

**Opposite:** Natural, organic, subdued

---

### 1.5 Glamorous Instability

**Meaning:** Beautiful but unstable, expensive but broken, diva energy

**Visual Markers:**
- Glossy, radiant surfaces
- Overlit beauty
- Glitter and sparkle
- One anchor holding on
- Feels expensive but fragile

**Usage:**
```
"glamorous instability"
"glamorous instability with overprint"
```

**Opposite:** Plain, stable, boring

---

### 1.6 Coquettish

**Meaning:** Playfully flirtatious, teasing, knowing

**Visual Markers:**
- Eyes preserved with knowing look
- Corruption feels like a wink
- Slight, intentional damage
- Pretty but with an edge
- Inviting but dangerous

**Usage:**
```
"coquettish glitter signal"
"coquettish with soft corruption"
```

**Opposite:** Serious, straightforward, blunt

---

### 1.7 Overstimulating

**Meaning:** Too much, sensory overload, beautiful chaos

**Visual Markers:**
- Maximum density
- Multiple drivers active
- Bright colors everywhere
- One anchor barely holding
- Feels like it's too much but you can't look away

**Usage:**
```
"overstimulating hyperpop rupture"
"overstimulating with pile-up delirium"
```

**Opposite:** Calm, minimal, restrained

## 4. Intensity Modifiers

### 4.1 Soft / Gentle

**Effect:** Reduces intensity, adds subtlety

**Usage:**
```
"soft seductive bloom"
"gentle pastel smear"
```

---

### 4.2 Hard / Harsh

**Effect:** Increases intensity, adds aggression

**Usage:**
```
"hard channel split"
"harsh compression damage"
```

---

### 4.3 Deep / Heavy

**Effect:** Increases density, adds weight

**Usage:**
```
"deep overprint stacking"
"heavy temporal echo"
```

---

### 4.4 Light / Airy

**Effect:** Reduces density, adds openness

**Usage:**
```
"light sparkle static"
"airy pastel glow"
```

---

### 4.5 Sharp / Crisp

**Effect:** Increases edge definition

**Usage:**
```
"sharp channel separation"
"crisp scanline contour"
```

---

### 4.6 Soft / Blurred

**Effect:** Reduces edge definition

**Usage:**
```
"soft bloom contamination"
"blurred overprint edges"
```
