# Subcategory System — Glitchcore

The full catalog of glitchcore branches, their artifact drivers, palette families, density ranges, emotional flavors, and common drift patterns.

## Purpose

This file defines the complete subcategory system for glitchcore — a catalog of distinct branches that share the same DNA (artifact-driven identity distortion, palette pressure, emotional consequence) but express it through different technical and emotional means.

This is the map. The other files are the territory.

---

## 1. The Twelve Core Subcategories

### 1.1 Candy Broadcast Hallucination

**Core Idea:**
Bright, saturated, pop-heavy glitch mode where channel splits, bloom, and chromatic collisions turn the figure into a candy-burn signal event.

**Main Artifact Drivers:**
- Channel split (primary)
- Bloom contamination
- Overprint stacking
- Color field collision

**Best Palette Families:**
- Hyperpop rupture
- Rainbow channel hallucination
- Gloss-electric glam
- Toxic cathode candy

**Best Density Range:**
- Level 3–5 (dense to pile-up delirium)

**Emotional Flavor:**
- Seductive
- Emotionally loud
- Synthetic euphoria
- Glamorous instability
- Sugary heartbreak

**Best Use Cases:**
- Bright portraits
- Beauty glitch
- Glam-pop faces
- Emotionally intense poster-like images
- Candy-colored identity fracture

**Common Drift:**
- Generic hyperpop editorial
- Random rainbow slop
- Empty pink/blue moodboard bullshit

## 2. Grouping by Major Behavior

### 2.1 Color / Identity Split Families

These rely heavily on chromatic instability and duplicated selfhood:
- Candy Broadcast Hallucination
- RGB Phantom
- Pastel Identity Smear

### 2.2 Maximal Stack Families

These thrive on layering and pressure:
- Composite Fever Dream
- Candy Broadcast Hallucination
- Candy-Crash Compression
- Glitter-Signal Overprint

### 2.3 Time / Body Families

These are about movement and repeated embodiment:
- Ghost-Frame Body
- RGB Phantom with phase lag
- Composite Fever Dream when temporal is present

### 2.4 Screen-Material Families

These make the display itself visible:
- CRT Contour
- Pixel Ghost
- Surveillance Apparition

### 2.5 Language / Interface Families

These use words and UI as emotional contamination:
- Text-Screen Heartbreak
- Surveillance Apparition
- Candy-Crash Compression with chat residue

### 2.6 Damage / Aggression Families

These feel more broken, stressed, or under attack:
- Raster Meltdown
- Candy-Crash Compression
- Surveillance Apparition
- Composite Fever Dream at high density

### 2.7 Glam / Beauty Families

These can support the more seductive pop-glitch branch:
- Pastel Identity Smear
- Candy Broadcast Hallucination
- Glitter-Signal Overprint
- RGB Phantom
- Candy-Crash Compression
- Text-Screen Heartbreak in glam mode

## 3. Fast Decision Guide

**If you want bright poppy face corruption:**
→ Candy Broadcast Hallucination

**If you want soft dreamy beauty damage:**
→ Pastel Identity Smear

**If you want hard chromatic spectral self-split:**
→ RGB Phantom

**If you want all the glitch at once but still hot:**
→ Composite Fever Dream

**If you want striped signal anatomy:**
→ CRT Contour

**If you want motion ghosts and repeated body copies:**
→ Ghost-Frame Body

**If you want more violent torn-feed damage:**
→ Raster Meltdown

**If you want low-res display-material body:**
→ Pixel Ghost

**If you want emotional phrases / chat residue / terminal sadness:**
→ Text-Screen Heartbreak

**If you want watched / institutional / CCTV mood:**
→ Surveillance Apparition

**If you want broken-export internet beauty:**
→ Candy-Crash Compression

**If you want glossy diva corruption with sparkle and layered light:**
→ Glitter-Signal Overprint

## 4. Summary Sentence

The glitchcore subcategory system organizes the style into distinct but hybrid-friendly branches, each defined by specific artifact behaviors, palette energies, density ranges, emotional flavors, and body/image transformation logics.

Don't just say "glitch." Say what kind of glitch, what kind of body, what kind of damage, what kind of color, and what kind of feeling.
