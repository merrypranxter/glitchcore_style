# Subcategories Reference Index

Master map of all glitchcore subtypes, branches, and hybrid categories.

## Purpose

This file is the master index for the entire glitchcore style system — every subtype, every branch, every hybrid, all in one place for quick reference.

This is the part that helps the app stop going:
"which glitch style should I use?"
and start going:
"candy broadcast = hyperpop bright, channel split + bloom, preserve eyes."

The index here is designed to be the starting point — the map that leads to all other files.

## 2. Hybrid Categories

### 2.1 Body + Contamination Hybrids

**Description:**
Body-based subtypes mixed with corruption drivers.

**Examples:**
- Ghost Frame + Compression
- Pastel Smear + Text Debris
- Candy Broadcast + Temporal Echo

**Logic:**
Body preservation + corruption overlay

---

### 2.2 Palette + Failure Hybrids

**Description:**
Bright palettes mixed with damage and degradation.

**Examples:**
- Candy Crash + Hyperpop Rupture
- RGB Phantom + Compression
- Glitter Signal + Codec Damage

**Logic:**
Beauty vs damage tension

---

### 2.3 Structure + Motion Hybrids

**Description:**
Structural drivers mixed with temporal/motion drivers.

**Examples:**
- CRT Contour + Ghost Frame
- Scanline + Temporal Echo
- Channel Split + Motion Trails

**Logic:**
Form + movement together
